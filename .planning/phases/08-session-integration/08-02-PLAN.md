**renderer.js — EEG waveform renderer:**

1. Add EEG constants at module top:
   - `const EEG_SAMPLE_RATE = 256;`
   - `const EEG_DISPLAY_SECONDS = 2;` (show last 2 seconds, matching buffer size)
   - `const EEG_DISPLAY_SAMPLES = EEG_SAMPLE_RATE * EEG_DISPLAY_SECONDS;` (512)
   - `const EEG_UV_RANGE = 100;` (display range: -100 to +100 µV, matching artifact threshold)
   - `const TP9_COLOR = '#3b82f6';` (blue — matches Neural Calm)
   - `const TP10_COLOR = '#8b5cf6';` (purple — distinct from TP9)

2. Add module state:
   - `let _eegCanvas = null, _eegCtx = null;`

3. Create `drawEEGWaveform()` function:
   - If `!_eegCtx` return
   - Get canvas dimensions (same dpr pattern as drawWaveform)
   - Clear canvas
   - Draw subtle center line at y=h/2 (rgba(255,255,255,0.05))
   - Draw channel labels: "TP9" top-left in TP9_COLOR, "TP10" top-right in TP10_COLOR (font 10px)
   - If `!AppState.museConnected`: draw flat lines at center for both channels (visual placeholder), return
   - Read EEG data from `AppState.eegBuffers[0]` (TP9) and `AppState.eegBuffers[3]` (TP10)
   - Read `AppState.eegHead` as the write pointer
   - For each channel (TP9 index 0, TP10 index 3):
     - Extract 512 samples in chronological order from circular buffer: `(eegHead - EEG_DISPLAY_SAMPLES + i + 512) % 512`
     - Map each sample to canvas Y: `y = h/2 - (sample / EEG_UV_RANGE) * (h/2 * 0.8)` (80% of half-height, leaves margin)
     - Draw as a continuous line path with 1.5px stroke width
     - TP9 uses TP9_COLOR, TP10 uses TP10_COLOR
   - Use stacked layout: TP9 in top half (y-center = h*0.3), TP10 in bottom half (y-center = h*0.7)
   - Draw a thin separator line at y=h/2 (rgba(255,255,255,0.08))
   - Each channel gets its own vertical range within its half

4. **Update startRendering() signature** to also accept EEG canvas:
   - New signature: `startRendering(waveformCanvas, spectrumCanvas, gaugeCanvas, pacerCanvas, sessionStartTime, sessionDuration, neuralCalmCanvas, eegCanvas)`
   - Store `_eegCanvas`, set up in `_setupAllCanvases()` with null guard

5. **Add to renderLoop():** Call `drawEEGWaveform()` after `drawNeuralCalmGauge()`

6. **Update stopRendering():** Clear EEG canvas if it exists
