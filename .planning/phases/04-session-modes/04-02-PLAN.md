**2. Create js/practice.js â€” Practice session controller:**

Module-level state:
- `_sessionStart` (Date.now timestamp)
- `_sessionDurationMs` (selected duration in ms)
- `_dspInterval` (setInterval handle)
- `_coherenceTrace` (array of coherence scores, 1 per second)
- `_chimePlayed` (boolean, prevents double chime)
- `_pausedForReconnect` (boolean)
- `_selectedDuration` (minutes, default 20)

Exports:
- `startPractice()` â€” Entry point. Reads `AppState.savedResonanceFreq`. Sets `AppState.sessionPhase = 'practice'`. Sets `AppState.pacingFreq = savedFreq`. Shows `#practice-session-viz`, hides placeholder. Calls `initAudio()`, `initDSP()`. Sets `_sessionStart = Date.now()`, `AppState.sessionStartTime = _sessionStart`. Resets `_coherenceTrace = []`, `_chimePlayed = false`. Calls `startRendering(practiceWaveformCanvas, null, practiceGaugeCanvas, practicePacerCanvas, _sessionStart, _selectedDuration * 60)` â€” pass `null` for spectrum canvas (no spectrum in practice). Calls `startPacer(savedFreq)`. Starts DSP tick interval (1s): `tick(elapsed)` then `_coherenceTrace.push(AppState.coherenceScore)`. Also checks for chime: if `elapsed >= _selectedDuration * 60` and `!_chimePlayed`, calls `playChime()`, sets `_chimePlayed = true`, updates "End Session" button text to "End Session" (maybe highlight it or add pulsing class).
- `stopPractice()` â€” Cleanup. Clears DSP interval. Calls `stopPacer()`, `stopRendering()`. Computes summary. Saves session. Shows summary section.
- `onDisconnect()` â€” If active: pause pacer, set `_pausedForReconnect`. Subscribe to `connected` for resume (named function, unsubscribe on resume/fail). On permanent fail: save partial session.
- `setDuration(minutes)` â€” Sets `_selectedDuration = minutes`. Updates active state of duration buttons.
- `initPracticeUI()` â€” Called from main.js on init. Sets up: subscribe to `savedResonanceFreq` to update frequency display, subscribe to `connected` to toggle start button disabled state, wire duration button clicks, wire start/end/done button clicks.

Internal functions:
- `_computeSummary()` â€” From `_coherenceTrace`: mean (rounded), peak (max), timeInHigh (count of scores >= 66), durationSeconds (trace.length or Date.now() - _sessionStart / 1000). Returns `{durationSeconds, mean, peak, timeInHigh, trace}`.
- `_showSummary(summary)` â€” Hides `#practice-session-viz`. Shows `#practice-summary`. Populates 4 metrics. Duration formatted as mm:ss. Time Locked In formatted as mm:ss.
- `_saveSession(summary)` â€” Calls `saveSession({mode: 'practice', date: new Date().toISOString(), durationSeconds: summary.durationSeconds, frequencyHz: AppState.pacingFreq, meanCoherence: summary.mean, peakCoherence: summary.peak, timeInHighSeconds: summary.timeInHigh, coherenceTrace: summary.trace})`.
- `_onDone()` â€” Hides summary. Shows placeholder. Sets `AppState.sessionPhase = 'idle'`.
