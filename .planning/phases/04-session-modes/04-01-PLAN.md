**3. Create js/discovery.js — the Discovery controller:**

Constants:
- `DISCOVERY_BLOCKS = [{bpm: 6.5, hz: 6.5/60}, {bpm: 6.0, hz: 6.0/60}, {bpm: 5.5, hz: 5.5/60}, {bpm: 5.0, hz: 5.0/60}, {bpm: 4.5, hz: 4.5/60}]`
- `BLOCK_DURATION_MS = 120000` (2 minutes)
- `INTER_BLOCK_PAUSE_MS = 4000` (4 seconds)

State machine phases: `'idle' | 'countdown' | 'block' | 'inter-block-pause' | 'comparison'`

Module-level state:
- `_phase`, `_blockIndex`, `_blockTimer`, `_blockResults` array, `_blockStartTime`, `_dspInterval`, `_sessionStart`, `_pausedForReconnect`, `_selectedIndex`

Exports:
- `startDiscovery()` — Entry point. Sets `AppState.sessionPhase = 'discovery'`. Shows session-viz, hides placeholder. Calls `initAudio()`, `initDSP()`. Sets `_sessionStart = Date.now()`, `AppState.sessionStartTime = _sessionStart`. Starts the DSP tick interval (1s). Shows a 3-2-1 countdown overlay, then calls `startBlock(0)`.
- `stopDiscovery()` — Cleanup. Clears all timers. Calls `stopPacer()`, `stopRendering()`. Sets `AppState.sessionPhase = 'idle'`. Shows placeholder, hides session-viz.
- `onDisconnect()` — If `_phase !== 'idle'`: stop pacer, set `_pausedForReconnect = true`. Subscribe to `connected` for resume (use named function ref, unsubscribe on resume or permanent fail).

Internal functions:
- `_showCountdown(seconds, onDone)` — Displays 3-2-1 countdown in the `#discovery-countdown` overlay. Uses `setInterval` to tick. Calls `onDone()` when done.
- `startBlock(index)` — Sets `_blockIndex = index`, `_phase = 'block'`. Calls `stopPacer()`, sets `AppState.pacingFreq = block.hz`, calls `startPacer(block.hz)`. Sets `_blockStartTime = Date.now()`, `AppState.sessionStartTime = _blockStartTime`. Updates rate label (`#discovery-rate-label`). Updates progress indicator (e.g., step dots: filled for completed, active for current, empty for upcoming). Calls `startRendering(waveformCanvas, spectrumCanvas, gaugeCanvas, pacerCanvas, _blockStartTime, BLOCK_DURATION_MS / 1000)` — countdown mode for 120s per block. Schedules `_blockTimer = setTimeout(() => endBlock(index), BLOCK_DURATION_MS)`.
- `endBlock(index)` — Captures metrics: `const elapsed = Math.min(120, (Date.now() - _blockStartTime) / 1000)`, `const hrSamples = getHRArray(elapsed)`, `const rsaAmplitude = computeRSAAmplitude(hrSamples)`, `const lfPower = AppState.lfPower`. Pushes to `_blockResults`. Calls `stopRendering()`, `stopPacer()`. If more blocks: calls `_showInterBlockPause(index)`. If last block: clears DSP interval, calls `_showComparison()`.
- `_showInterBlockPause(completedIndex)` — Sets `_phase = 'inter-block-pause'`. Shows the `#discovery-pause-overlay` with: "Block {N}/5 complete", "Next: {nextBpm} breaths/min", and a 4-3-2-1 countdown. After countdown, hides overlay, calls `startBlock(completedIndex + 1)`.
- `_showComparison(results)` — Sets `_phase = 'comparison'`. Hides session-viz. Shows `#discovery-comparison` section. Finds best index (max RSA amplitude). Sets `_selectedIndex = bestIndex`. Draws bar chart on `#comparison-chart-canvas`. Adds click listener on canvas for override selection. Shows "Recommended" label on best bar. Shows Confirm button.
- `_drawComparisonChart(canvas, results, selectedIndex)` — Canvas 2D bar chart. X-axis: 5 frequencies (BPM labels). Y-axis: RSA amplitude. Selected bar in full teal (#14b8a6), others dimmed. "Recommended" label above auto-selected bar. Use DPR-aware sizing.
- `_onConfirm()` — Saves: `await setSetting('resonanceFreq', selectedHz)`, `AppState.savedResonanceFreq = selectedHz`. Saves discovery session: `await saveSession({mode: 'discovery', date: new Date().toISOString(), selectedFreqHz: selectedHz, blocks: _blockResults})`. Sets `_phase = 'idle'`, `AppState.sessionPhase = 'idle'`. Shows success message briefly, then returns to placeholder with updated text ("Resonance frequency set: {bpm} breaths/min").

BLE disconnect handling: Named function `_onConnectedChange(value)` — if disconnected during active block: pause pacer, show reconnecting banner (`#connection-banner`), set `_pausedForReconnect = true`. If reconnected and `_pausedForReconnect`: resume pacer. If `showManualReconnect` becomes true (permanent fail): call `stopDiscovery()` and save partial results. Subscribe at startDiscovery, unsubscribe at stopDiscovery.
