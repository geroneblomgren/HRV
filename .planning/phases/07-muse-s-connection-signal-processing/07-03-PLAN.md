<objective>
Build the EEG signal processing pipeline that computes the Neural Calm score from Muse-S EEG data. This pipeline runs a sliding-window FFT on TP9/TP10 channels, extracts alpha (8-12 Hz) and beta (13-30 Hz) band power, rejects artifact-contaminated epochs, normalizes against a per-session baseline, and produces a 0-100 Neural Calm score updating twice per second. Also includes eyes-open detection based on sudden alpha power drops.

Purpose: This is the brain-state biofeedback metric — it tells the user how calm their neural activity is during breathing practice, displayed alongside HRV coherence.

**1. EEG FFT Constants**

```javascript
const EEG_FFT_SIZE = 512;          // 2 seconds at 256 Hz
const EEG_UPDATE_INTERVAL = 128;   // new samples between FFT runs (0.5 sec)
const ALPHA_LOW = 8, ALPHA_HIGH = 12;
const BETA_LOW = 13, BETA_HIGH = 30;
const EEG_FS = 256;
const ARTIFACT_THRESHOLD_UV = 100; // peak-to-peak µV for epoch rejection
const BASELINE_EPOCHS_NEEDED = 10; // 10 x 2-sec = 20 sec baseline window
const ALPHA_DROP_THRESHOLD = 0.40; // 40% drop triggers eyes-open indicator
```

**2. EEG State Variables**

```javascript
let _eegFft = null;                // new FFT(512) — separate from dsp.js instance
let _newSampleCount = 0;           // samples since last FFT
let _baselineRatios = [];          // alpha/(alpha+beta) during baseline collection
let _recentRatios = [];            // last 10 ratios for eyes-open detection
let _lastValidNeuralCalm = 0;     // carried forward during artifact rejection
let _eyesOpenTimer = null;
```

**3. initEEGPipeline()**

- Create `_eegFft = new FFT(EEG_FFT_SIZE)` — a separate instance from the HRV FFT in dsp.js
- Reset all EEG state variables
- Register EEG callback via `setEEGCallback(handleEEGSamples)` from MuseAdapter
- Set `AppState.eegCalibrating = true`

**4. handleEEGSamples(channelIndex, samples)**

This is called by MuseAdapter for each EEG channel notification (12 samples).

- Only process channels 0 (TP9) and 3 (TP10) — these are the channels used for Neural Calm per research finding that TP9/TP10 are superior to AF7/AF8 on Muse due to Fpz reference proximity
- Track sample count. Only from channel 0 (TP9): increment `_newSampleCount` by 12
- When `_newSampleCount >= EEG_UPDATE_INTERVAL` (128 samples = 0.5 sec): trigger `_computeNeuralCalm()`

**5. _computeNeuralCalm()**

For each of TP9 (index 0) and TP10 (index 3):

a. **Extract epoch** — Read 512 samples from `AppState.eegBuffers[channelIndex]` in chronological order using the circular buffer pattern: `buffer[(eegHead - EEG_FFT_SIZE + i + EEG_FFT_SIZE*100) % EEG_FFT_SIZE]` for i in 0..511.

b. **Artifact check** — Find min and max of the 512-sample epoch. If `(max - min) > ARTIFACT_THRESHOLD_UV` (100 µV), reject this epoch. If EITHER TP9 or TP10 fails, reject the entire computation and carry forward `_lastValidNeuralCalm`. Return early.

c. **Hann window** — Apply: `windowed[i] = epoch[i] * (0.5 - 0.5 * Math.cos(2 * Math.PI * i / EEG_FFT_SIZE))`

d. **FFT** — Run `_eegFft.realTransform(complexOut, windowed)`. Output is interleaved [re0, im0, re1, im1, ...].

e. **Band power integration** — For alpha and beta bands:
```javascript
function integrateBand(complexOut, lowHz, highHz) {
  const binLow  = Math.round(lowHz  * EEG_FFT_SIZE / EEG_FS);
  const binHigh = Math.round(highHz * EEG_FFT_SIZE / EEG_FS);
  let sum = 0;
  for (let i = binLow; i <= binHigh; i++) {
    const re = complexOut[2 * i], im = complexOut[2 * i + 1];
    sum += re * re + im * im;
  }
  return sum;
}
```

f. **Average across TP9 and TP10** — Compute alpha and beta power for each channel, then average: `alphaPower = (alphaTP9 + alphaTP10) / 2`, same for beta.

g. **Ratio** — `ratio = alphaPower / (alphaPower + betaPower + 1e-10)`

h. **Baseline normalization** (per-session relative):
- During baseline collection (`_baselineRatios.length < BASELINE_EPOCHS_NEEDED`): push ratio, set `AppState.neuralCalm = 0`, set `AppState.rawNeuralCalmRatio = ratio`. Return.
- After baseline: `baselineMean = average(_baselineRatios)`. Normalize: `normalized = clamp(0, 1, (ratio - baselineMean * 0.5) / baselineMean)`. Set `AppState.neuralCalm = Math.round(normalized * 100)`.
- When baseline collection completes (exactly at BASELINE_EPOCHS_NEEDED), set `AppState.eegCalibrating = false`.

i. **Eyes-open detection** — After computing ratio, call `_checkEyesOpen(ratio)`:
- Maintain `_recentRatios` (last 10 values, FIFO)
- If fewer than 5 values, skip
- Compute mean of all but last value
- If `(mean - ratio) / (mean + 1e-10) > ALPHA_DROP_THRESHOLD`: set `AppState.eyesOpenWarning = true`, clear existing timer, set 3-second timer to reset to false

j. **Store** — `AppState.rawNeuralCalmRatio = ratio`, `_lastValidNeuralCalm = AppState.neuralCalm`

**6. stopEEGPipeline()**

- Call `setEEGCallback(null)` to deregister
- Clear `_eyesOpenTimer`
- Reset `AppState.neuralCalm = 0`, `AppState.eegCalibrating = true`, `AppState.eyesOpenWarning = false`
- Reset state variables

export function stopEEGPipeline() { ... }
