**renderer.js changes:**

1. Add import at top: `import { getAudioTime } from './audio.js';`

2. Add module-level variables:
   - `_pacerCanvas`, `_pacerCtx` — for the breathing circle canvas
   - `_prevPhase = 'inhale'` — tracks previous phase for label fade transition
   - `_labelOpacity = 1.0` — for Inhale/Exhale text fade
   - `_sessionDuration = 0` — total session duration in seconds (0 = no countdown)

3. Add `drawBreathingCircle()` function (called from renderLoop):

   **Phase position calculation (CRITICAL — use AudioContext.currentTime for sync):**
   ```
   const audioTime = getAudioTime();
   if (audioTime === 0) return; // audio not started yet
   const halfPeriod = 1 / (AppState.pacingFreq * 2);
   const currentPhaseStart = AppState.nextCueTime - halfPeriod;
   const elapsed = audioTime - currentPhaseStart;
   const t = Math.max(0, Math.min(1, elapsed / halfPeriod));
   ```
   This uses the same clock as the audio scheduler. Do NOT use Date.now() or performance.now().

   **Easing:** Apply smoothstep `eased = t * t * (3 - 2 * t)` — more deliberate than pure sine at endpoints.

   **Radius mapping:**
   - Determine if currently inhaling: `const isInhale = AppState.nextCuePhase === 'exhale';` (nextCuePhase is the NEXT phase, so if next is exhale, we're currently inhaling)
   - minRadius = min(w, h) * 0.15, maxRadius = min(w, h) * 0.35
   - inhale: radius = minR + (maxR - minR) * eased (expanding)
   - exhale: radius = maxR - (maxR - minR) * eased (contracting)

   **Draw glowing teal ring:**
   - ctx.save()
   - ctx.shadowColor = '#14b8a6', ctx.shadowBlur = 20 + eased * 15 (glow pulses with expansion)
   - ctx.beginPath(), ctx.arc(cx, cy, radius, 0, Math.PI * 2)
   - ctx.strokeStyle = '#14b8a6', ctx.lineWidth = 3, ctx.stroke()
   - ctx.shadowBlur = 0 (reset shadow BEFORE drawing text)
   - ctx.restore()

   **Phase label (Inhale/Exhale) inside circle:**
   - Detect phase change: if current phase differs from _prevPhase, set _labelOpacity = 0, update _prevPhase
   - Lerp _labelOpacity toward 1.0: `_labelOpacity = Math.min(1.0, _labelOpacity + 0.06)` (~17 frames = ~280ms fade-in)
   - Draw text: ctx.fillStyle = `rgba(232, 232, 232, ${_labelOpacity})`, font = `${Math.round(min(w,h) * 0.05)}px system-ui, sans-serif`, textAlign 'center', textBaseline 'middle'
   - Text = isInhale ? 'Inhale' : 'Exhale', positioned at (cx, cy - min(w,h)*0.03)

   **Countdown timer inside circle (below phase label):**
   - If _sessionDuration > 0: compute remaining = _sessionDuration - (Date.now() - AppState.sessionStartTime) / 1000. Wall clock is fine for countdown display (not timing-critical).
   - If _sessionDuration is 0, show elapsed time instead (for discovery blocks where countdown is set per-block by Phase 4)
   - Format as mm:ss. Draw below label at (cx, cy + min(w,h)*0.04), font = `${Math.round(min(w,h) * 0.04)}px monospace`, color rgba(232, 232, 232, 0.7)

4. Update `renderLoop()` to call `drawBreathingCircle()` as the FIRST draw call (before waveform, spectrum, gauge).

5. Update `startRendering()` signature: add `pacerCanvas` parameter and `sessionDuration` parameter (default 0). Set up `_pacerCanvas` and `_pacerCtx` via setupCanvas(). Store `_sessionDuration`.

6. Update `stopRendering()` to clear the pacer canvas too.
