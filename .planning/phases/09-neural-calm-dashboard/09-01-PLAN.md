- Compute average, display with 1 decimal in `#dash-calm-7d`
- Show '--' if no Neural Calm sessions exist

**js/dashboard.js — Neural Calm trend line (in `_drawChart`, after coherence dots):**
Draw a blue line connecting days that have `meanNeuralCalm !== null`:
- Filter `sessSlice` to entries where `meanNeuralCalm !== null` — call this `calmSlice`
- If `calmSlice.length > 0`, draw using the same smooth quadratic curve technique as HRV line
- Use `cohYPx()` for Y mapping (Neural Calm is 0-100 same as coherence)
- Line color: #3b82f6 (Tailwind blue-500, matches Neural Calm session gauge)
- Line width: 1.5 (thinner than HRV's 2 to avoid visual clutter)
- Line opacity: 0.85 via globalAlpha (slightly transparent to layer cleanly under coherence dots)
- Reset globalAlpha to 1.0 after drawing
- Gap handling: use **broken line** (moveTo for each segment start after a gap day). Iterate chronologically; when consecutive days both have data, quadraticCurveTo. When there is a gap, start a new sub-path with moveTo. This means the line breaks where there is no Muse-S data — honest representation.
- Add small circle markers (radius 3) at each Neural Calm data point for hover targeting
- Push each calm point into `_hitTargets` with `type: 'calm'`
