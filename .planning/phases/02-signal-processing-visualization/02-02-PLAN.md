Create js/renderer.js as an ES module with these components:

**Imports:**
```javascript
import { AppState } from './state.js';

```

**Constants:**
- HR_MIN = 40, HR_MAX = 120 (fixed Y-axis range per user decision)
- WAVEFORM_WINDOW_SECONDS = 60 (60s visible window — shows ~6 RSA cycles at resonance breathing)
- MAX_FREQ_HZ = 0.5 (spectrum X-axis range)
- LF_LOW = 0.04, LF_HIGH = 0.15 (LF band shading boundaries)
- SAMPLE_RATE = 4, FFT_SIZE = 512 (for binToHz conversion)
- TEAL = '#14b8a6' (accent teal — the primary chart color)
- TEAL_FILL_TOP = 'rgba(20, 184, 166, 0.5)'
- TEAL_FILL_BOTTOM = 'rgba(20, 184, 166, 0.02)'
- ZONE_COLORS = { low: '#ef4444', building: '#eab308', high: '#22c55e' }
- ZONE_THRESHOLDS = { building: 31, high: 66 } (per research recommendations)

**Module state:**
- _rAF = null (animation frame ID)
- _waveformCanvas, _waveformCtx
- _spectrumCanvas, _spectrumCtx
- _gaugeCanvas, _gaugeCtx
- _displayedScore = 0 (for smooth coherence interpolation)
- _pulsePhase = 0 (for high-zone ring pulse)
- _sessionStartTime = null (for calibration countdown)
- _calibrationFadeAlpha = 0 (for fade-in transition when calibration ends)

**startRendering(waveformCanvas, spectrumCanvas, gaugeCanvas, sessionStartTime):**
1. Store canvas references and get 2D contexts
2. Set canvas dimensions to match their CSS layout size (handle devicePixelRatio for crisp rendering)
3. Store sessionStartTime for calibration countdown
4. Reset _displayedScore = 0, _pulsePhase = 0, _calibrationFadeAlpha = 0
5. Start rAF loop via requestAnimationFrame(renderLoop)

**stopRendering():**
1. cancelAnimationFrame(_rAF)
2. Clear all canvases

**renderLoop(timestamp):**
1. Schedule next frame: _rAF = requestAnimationFrame(renderLoop)
2. Call drawWaveform()
3. Call drawSpectrum()
4. Call drawCoherenceGauge()

**drawWaveform():**
Per CONTEXT.md locked decisions — filled area plot, teal/cyan line + fill, dark background, fixed Y-axis 40-120 BPM, renders from session start including during calibration.

1. Get HR data: `const hrData = getHRArray(WAVEFORM_WINDOW_SECONDS)`
2. If hrData is empty, draw empty axes and return
3. clearRect the entire canvas
4. Draw Y-axis grid lines at 60, 80, 100 BPM with very subtle lines (rgba white 0.05)
5. Create vertical linear gradient: teal at top (0.5 opacity) fading to near-transparent at bottom
6. Build path: for each HR sample, map BPM to Y pixel using fixed 40-120 range, lineTo
7. Close path to baseline, fill with gradient
8. Redraw the line on top with solid teal, lineWidth 2
9. Draw subtle Y-axis labels (60, 80, 100) in small monospace text, rgba white 0.3

**drawSpectrum():**
Per CONTEXT.md locked decisions — filled area plot, LF band highlighted with background shading, peak marked with dot and frequency label.

1. clearRect
2. Draw LF band background shading: fillRect from 0.04 Hz to 0.15 Hz, full height, rgba teal 0.07 — always visible even during calibration
3. If AppState.calibrating:
   - Compute secondsRemaining = Math.max(0, 120 - sessionElapsed)
   - Draw "Calibrating..." centered text with countdown in subtle white
   - Draw small progress bar below text
   - Return (no spectrum data yet)
4. Handle fade-in: if _calibrationFadeAlpha < 1, increment by 0.02 per frame, use as globalAlpha
5. Read AppState.spectralBuffer — if null, return
6. Normalize PSD: find max value, scale so peak = 80% of canvas height
7. Build filled area path over frequency axis (0 to 0.5 Hz)
8. Fill with teal vertical gradient (same style as waveform)
9. Find peak bin in LF range, draw small dot (radius 4) at peak position
10. Draw peak frequency label (e.g., "0.10 Hz") in small monospace text
11. Draw X-axis labels at 0.04, 0.10, 0.15, 0.25 Hz in monospace (per research recommendation)

Helper: binToHz(bin) = bin * SAMPLE_RATE / FFT_SIZE

**drawCoherenceGauge():**
Per CONTEXT.md locked decisions — number inside progress ring, color zones (red/yellow/green), labels ("Low"/"Building"/"Locked In"), pulse animation in high zone, smooth transitions.

1. Determine zone from AppState.coherenceScore: < 31 = 'low', < 66 = 'building', >= 66 = 'high'
2. If AppState.calibrating:
   - Draw grey ring at 0 with "Calibrating..." text
   - Draw countdown and progress bar (same as spectrum)
   - Return
3. Handle fade-in (same _calibrationFadeAlpha as spectrum)
4. Smooth interpolation: _displayedScore += (AppState.coherenceScore - _displayedScore) * 0.08
5. Get zone color from ZONE_COLORS
6. Draw background ring (full circle, rgba white 0.1, lineWidth 12)
7. Draw filled arc: sweep from -PI/2 proportional to _displayedScore/100
8. If high zone: modulate lineWidth with sine wave (_pulsePhase += 0.05, lineWidth = 12 + sin * 2.5) — subtle pulse animation per user decision
9. Draw score number centered in ring, bold, sized relative to radius
10. Draw zone label below number ("Low" / "Building" / "Locked In")
11. Both number and ring use the zone color

**Anti-patterns to avoid:**
- Do NOT create separate rAF loops per renderer (one shared loop)
- Do NOT use setInterval for rendering (rAF only)
- Do NOT auto-scale the waveform Y-axis (fixed 40-120 BPM per user decision)

2. Add DSP tick interval management:
```javascript
let _dspInterval = null;
let _sessionStart = null;

function startSession() {
  _sessionStart = Date.now();
  initDSP();

  // Get canvas elements
  const waveformCanvas = document.getElementById('waveform-canvas');
  const spectrumCanvas = document.getElementById('spectrum-canvas');
  const gaugeCanvas = document.getElementById('gauge-canvas');

  // Start rendering
  startRendering(waveformCanvas, spectrumCanvas, gaugeCanvas, _sessionStart);

  // Start DSP tick at 1-second interval (use setInterval, NOT rAF — DSP should run even when tab hidden)
  _dspInterval = setInterval(() => {
    const elapsed = Math.floor((Date.now() - _sessionStart) / 1000);
    tick(elapsed);
  }, 1000);

  // Show session viz, hide placeholder
  document.querySelector('.session-viz').classList.add('active');
  document.getElementById('discovery-placeholder').style.display = 'none';
}

function stopSession() {
  if (_dspInterval) {
    clearInterval(_dspInterval);
    _dspInterval = null;
  }
  stopRendering();
  document.querySelector('.session-viz').classList.remove('active');
  document.getElementById('discovery-placeholder').style.display = '';
}
```
