Modify practice.js to insert a tuning phase before every practice session.

**New import:**

**Modify `startPractice()` to run tuning first:**

The existing startPractice() immediately starts the session. Change it to:

1. Early checks remain the same (if _active return, check savedFreq — but for tuning, allow null savedFreq since tuning handles first-session case)
2. Set `_active = true`
3. Show the tuning overlay (`tuning-overlay`), hide placeholder and summary
4. Initialize audio and DSP: `initAudio()`, `initDSP()`
5. Start the tuning scanning renderer: `startTuningRenderer(tuningRingCanvas)`
6. Start a 1-second interval that updates the tuning-time-remaining text (60 - elapsed seconds) and tuning-current-freq text with AppState.tuningCurrentFreqBPM + " BPM"
7. `const result = await startTuning(AppState.savedResonanceFreq)`
8. Stop tuning renderer: `stopTuningRenderer()`
9. Clear the time-remaining interval
10. Hide tuning overlay, show tuning result:
    - Set `tuning-result-freq` text: `Today: ${result.freqBPM.toFixed(1)} BPM`
    - If AppState.tuningStoredFreqBPM > 0 (not first session):
      - Set comparison text: `Previously: ${AppState.tuningStoredFreqBPM.toFixed(1)} BPM`
      - If shift > 0.3 BPM: Show celebration div with text:
        `Your resonance shifted from ${stored} to ${result.freqBPM.toFixed(1)} BPM — improved vagal tone`
        (Use the arrow character for downward shift direction if result < stored, indicating improved vagal tone per the design spec)
    - Else (first session): Set comparison text: `First tuned session`
11. Show tuning-result div
12. Wait 3 seconds (4 seconds if celebration is shown, for readability)
13. Hide tuning-result div
14. **Now start the actual practice session** using result.freqHz as the pacing frequency:
    - Set `AppState.pacingFreq = result.freqHz`
    - Set `AppState.savedResonanceFreq = result.freqHz` (update stored freq for next session)
    - Continue with existing session start logic: show session-viz, start rendering, start pacer at result.freqHz, start DSP tick, etc.

**Make startPractice async** — change `export function startPractice()` to `export async function startPractice()`.
